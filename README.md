# Alzheimer's Support Chatbot

A supportive chatbot that provides **general information and caregiver guidance** for
Alzheimer's disease and dementia. It is built by fine-tuning a GPT-2 language model on
Alzheimer's-related question–answer data, with a basic medical-safety layer and a
Streamlit chat interface.

>  **Disclaimer**
> This chatbot is for **general education and caregiver support only**. It does **not**
> diagnose disease, prescribe treatment, or replace a qualified healthcare professional.
> For emergencies, severe symptoms, or medication changes, contact a doctor or emergency
> services. This project is **not a clinically validated medical device**.

---

## Features

- Fine-tuned GPT-2 model trained on Alzheimer's Q&A data
- Safety layer that detects crisis, urgent-medical, medication, and diagnosis questions
  and responds with appropriate guidance
- Automatic removal of fabricated/broken links from generated answers
- Automatic medical disclaimer appended to responses
- Streamlit web GUI with a disclaimer banner, starter questions, and conversation export
- Optional FastAPI endpoint for programmatic access
- One-click `.bat` launcher for Windows

## Requirements

- Python 3.11
- An NVIDIA GPU is recommended for training (CPU works but is slow)
- Key libraries: PyTorch, Transformers, scikit-learn, pandas, ftfy, Streamlit, FastAPI

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Alzheimer_chatbot.git
cd Alzheimer_chatbot
```

### 2. Create and activate an environment

Using conda:

```bash
conda create -n alzheimer python=3.11 -y
conda activate alzheimer
```

### 3. Install PyTorch (GPU build)

For NVIDIA GPUs (adjust the CUDA version to match your driver):

```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu124
```

For CPU only:

```bash
pip install torch torchvision
```

### 4. Install the remaining requirements

```bash
pip install -r requirements.txt
```

## Usage

Run these from the project root so the `src.` imports resolve.

### Step 1 — Prepare the data

Place your CSV files in the `data/` folder, then:

```bash
python 01_prepare_data.py
```

This cleans and combines the CSVs and writes processed files to `data/processed/`.

### Step 2 — Train the model

```bash
python 02_train_model.py --batch-size 8 --gradient-accumulation-steps 2 --epochs 3
```

The trained model is saved to `models/alzheimer-gpt2/`.
(The base model is set in `src/config.py` via `BASE_MODEL_NAME`.)

### Step 3 — Test in the terminal

```bash
python 03_test_chatbot.py
```

### Step 4 — Launch the GUI

```bash
streamlit run src/gui.py --server.fileWatcherType none
```

On Windows you can instead double-click **`Start_Chatbot.bat`**.
The app opens at `http://localhost:8501`.

### (Optional) Run the API

```bash
python 04_run_api.py
```

Then open `http://127.0.0.1:8000/docs`.

## Configuration

All settings live in `src/config.py`, including:

- `BASE_MODEL_NAME` — base model to fine-tune (e.g. `distilgpt2`, `gpt2`, `gpt2-medium`)
- `MAX_NEW_TOKENS`, `TEMPERATURE`, `TOP_P`, `REPETITION_PENALTY` — generation settings
- File paths for data and models

## Notes & Limitations

- The model is a fine-tuned GPT-2. It generates fluent text but **can produce incorrect
  or fabricated information**. Always verify medical details with a professional.
- The safety layer is keyword-based and is **not** a substitute for clinical review.
- Trained model files and processed data are **not** included in the repository (they are
  large and are regenerated locally). Run Steps 1–2 to produce them.
